# Challenge 1 Rovera Alura

### Propósito del análisis

El objetivo de este proyecto es analizar los datos de ventas de Alura Store para identificar patrones de compra, comportamiento de clientes y oportunidades de mejora en el negocio.
A través de técnicas de análisis exploratorio de datos (EDA) se buscó transformar datos en información estratégica que permita apoyar la toma de decisiones.

### Ejemplos de análisis realizados

Durante el análisis se exploraron diferentes aspectos de los datos, como:

-Distribución de ventas por categoría de producto.

-Relación entre el método de pago y el valor de compra.

-Productos más vendidos.

-Tendencias de ventas.

### Insights obtenidos

Algunos hallazgos importantes del análisis fueron:

-Ciertas categorías de productos concentran la mayor parte de las ventas.

-Algunos métodos de pago están asociados con compras de mayor valor.

-Existen productos con alta frecuencia de compra que podrían ser estratégicos para promociones.

-Estos insights pueden ayudar a optimizar estrategias comerciales y mejorar la experiencia del cliente.
